## 🔗 Linko: 링크를 저장하고 관리하는 당신의 보관함

Linko는 사용자들이 **웹 링크를 효율적으로 저장, 분류 및 관리**할 수 있도록 설계된 서비스입니다. 중요한 정보, 흥미로운 기사, 자주 방문하는 페이지 등을 한 곳에 모아 시각적으로 쉽게 접근할 수 있게 돕습니다.

---

### 1. 🚪 사용자 접근 (Access)

사용자가 서비스에 접속하고 인증/인가를 거쳐 계정을 관리하는 부분입니다. 모든 인증은 **Firebase Authentication**을 통해 안전하게 처리됩니다.

#### 1.1. 랜딩 페이지 (Landing Page)

* **서비스 소개:** 서비스 이름 **"Linko"**와 핵심 목적, 주요 장점을 명확하게 제시합니다.
* **서비스 진입 유도:** `로그인` 버튼 및 중앙의 `Linko 시작하기` 버튼을 통해 서비스 진입을 유도합니다.

#### 1.2. 로그인 페이지

* **인증 필드:** 이메일과 비밀번호 입력 필드.
* **인증 처리:** Firebase Authentication을 사용한 인증 절차를 거칩니다.
* **신규 사용자 유도:** `계정이 없으신가요? 회원가입` 링크를 제공합니다.

#### 1.3. 회원가입 페이지

* **입력 필드:** 이메일, 비밀번호, 비밀번호 확인 입력 필드.
* **계정 생성:** Firebase Authentication을 통해 새로운 계정을 생성합니다.

---

### 2. 🏠 핵심 기능: 메인 페이지 및 링크 관리

로그인한 사용자를 위한 Linko의 핵심 작업 공간입니다.

#### 2.1. 메인 페이지 (링크 보관함)

사용자가 저장한 모든 링크를 한눈에 볼 수 있는 대시보드 역할을 합니다.

* **편의 기능:** 상단에 검색창, 오른쪽 상단에 로그아웃 버튼 제공.
* **링크 분류:** '즐겨찾기'와 같은 특정 기준으로 분류된 섹션/폴더를 지원합니다.
* **시각화:** 각 링크는 제목/내용과 함께 이미지 썸네일로 표시됩니다.
* **링크 추가 접근:** 우측 하단에 `+` (추가) 버튼을 배치하여 신규 링크 저장을 쉽게 할 수 있습니다.

#### 2.2. 링크 추가 (Link Add)

새로운 링크를 저장하는 기능을 제공하는 모달 또는 페이지입니다.

| 필드 명 | 용도 |
| :--- | :--- |
| **제목** | 링크 식별 이름 |
| **썸네일 URL** | 링크의 시각적 대표 이미지 URL |
| **링크 (URL)** | 저장하고자 하는 실제 웹페이지 URL |

* **데이터 저장:** `저장` 버튼 클릭 시, 정보는 **Firestore 데이터베이스**에 저장되어 메인 페이지에 표시됩니다.

---

### 3. ✨ 부가 기능: 프리미엄 안내

서비스의 추가적인 수익 모델 또는 고급 기능을 안내하는 페이지입니다.

* **플랜 비교:** 무료/유료 구독 플랜의 혜택을 비교 제시합니다.
* **구독 옵션:** 1개월 무료, 1개월, 1년 등 다양한 구독 기간 옵션과 각 기간별 가격이 명시됩니다 (예: 1개월 5,000원).
* **목적:** 저장 공간 무제한, 고급 검색 기능 등 추가적인 혜택을 제공할 계획입니다.

---

### 🛠️ 사용 기술 스택 

* **백엔드/인프라:** Firebase Platform
    * **인증:** Firebase Authentication
    * **데이터베이스:** Firestore
    * **호스팅:** Firebase Hosting (kalguksiu.web.app 기반)
